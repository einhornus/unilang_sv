Pronunciation

  • IPA(key): /soː/
  • audio

Etymology 1

From Old Norse svá, from Proto-Germanic *swa, *swē.

Adverb

så

  1. so, thus, like that, in that way, in a certain manner
    Är det inte så?
    Isn't it so?
    så kallad
    so called
    så det så!
    so there!
  2. so, that, to that degree, to a certain extent
    jag trodde inte den var så stor
    I didn't think it was that big
    den är så stor att inte får plats
    it is so big that it doesn't fit
  3. how, what
    så roligt!
    how nice!
    så synd!
    what a pity!
    så tråkigt!
    how sad!
  4. then, thereafter
    se först åt vänster, så åt höger
    first look left, then right
  5. (slang) so; very

Conjunction

så

  1. so, that
    så att
    so that
    så att inte
    lest
    det regnade, så vi satte oss inomhus
    it rained, so we took a table indoors

Pronoun

så

  1. so, that
    i så fall
    if so, in that case

Etymology 2

From Old Norse sár, from Proto-Germanic *saihaz.

Noun

så c

  1. a tub, especially one for animals to drink from
