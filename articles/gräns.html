gräns (6191)


Etymology

From German Grenze, from Proto-Slavic *granica (“boundary, border”).

Pronunciation

  • audio
    		(file)

Noun

gräns c

  1. bound, boundary; the border of a territory
  2. (mathematics) bound; a value which is larger (or smaller) than every element in a given set
  3. border; where one country ends and the next begins
  4. limit; restriction which may not be crossed
