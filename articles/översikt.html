översikt (36392)


Noun

översikt c

  1. overview
  2. site map

Usage notes

This word is recommended as the translation of site map by Svenska datatermgruppen, an organisation which is working on establishing a comprehensive set of Swedish computer related words. However, many other words are in common use, such as:

  • sidöversikt
  • sajtöversikt / siteöversikt
  • sitemap
  • sajtkarta / sitekarta
  • webbkarta / webkarta
  • webböversikt
  • webbplatsöversikt
